# Portfolio Website

A clean single-page portfolio built with plain HTML, CSS, and JavaScript.

## Project Structure

- `src/index.html` - Page structure and content
- `src/style.css` - Theme, layout, responsive styles, animations
- `src/script.js` - Interactions (typing effect, theme toggle, mobile menu, scroll progress, reveal animations)

## Run Locally

Open `src/index.html` in your browser.

## Features

- Responsive layout
- Dark/light theme with persistence
- Scroll progress indicator
- Typing animation in hero section
- Reveal-on-scroll section animation
- Mobile navigation menu

## Customize

- Update text/content in `src/index.html`
- Update skill list and project cards in `src/index.html`
- Change colors in `:root` variables in `src/style.css`
- Update typed roles in `roles` array in `src/script.js`
